About application

Django project to show current weather and 5 day forecast in the city.

Project uses OpenWeatherMap API for gathering current and forecast weather data.


The following technologies were used in the project

  • Django 3.0
  • Python 3.6 - including Pandas, NumPy, MatPlotLib
  • Bootstrap 4 (CSS, JS)
